# Smart To-Do Manager

A desktop-based task management application developed using Python and Tkinter.

## Features
- Add, delete and complete tasks
- Priority management (High, Medium, Low)
- Due date tracking
- Live task search
- Progress bar
- Analytics dashboard using Matplotlib and Seaborn
- JSON-based persistent storage

## Tech Stack
- Python
- Tkinter
- JSON
- Pandas
- Matplotlib
- Seaborn

## How to Run

1. Install the required libraries

```bash
pip install pandas matplotlib seaborn
```

2. Run

```bash
python todo_app.py
```
